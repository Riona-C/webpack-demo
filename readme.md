### webpack基础配置

### author: 小橙子

### 安装依赖
```
npm install
```

### 打包并启动监听
```
npm run watch
```

### 自定义服务器脚本打包
```
npm run server
```

### 启动本地服务
```
npm run dev
```

### 打包js，es6转es5语法需借助一下依赖
```
babel-loader、@babel/polyfill、@babel/preset-env
```

### 打包react的jsx语法需要借助
```
@babel/preset-react
```
